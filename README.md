# EqualCard

[イコールカード｜算数に強くなる教育パズルカード教材](http://equalcard.co.jp/)

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'equal_card'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install equal_card

## Usage

```
$ equal_card answer '45643*-2'
```

```
カードは 45643*-2
4=34-5*6
4=34-6*5
54*6=324
5*6=34-4
6*54=324
6*5=34-4
34-5*6=4
34-6*5=4
34-4=5*6
34-4=6*5
324=54*6
324=6*54
```
